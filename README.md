# 🟣 Solana Wallet Generator

High-entropy Solana (SPL) wallet generator using Ed25519 curve derivation.

## 🚀 How to Run
1. **Library Install:** `pip install -r requirements.txt`
2. **Execution:** `python sol.py`
3. **Usage:** Follow the colored prompts in the terminal.

## 🛠 Technical Specifications
| Feature | Value |
| :--- | :--- |
| **Algorithm** | Ed25519 |
| **Derivation Path** | `m/44'/501'/0'/0/0` |
| **Output File** | `solwallet.txt` |
